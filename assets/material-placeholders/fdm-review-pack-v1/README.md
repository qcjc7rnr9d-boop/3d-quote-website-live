# FDM Material Placeholder Review Pack v1

Review-only material images for the FDM material library. These are intended to become locked library defaults while still allowing store admins to override images per material.

## How to review

- Open `review.html` in a browser for a searchable visual gallery.
- Use `contact-sheets/` for fast family-level inspection.
- The authoritative machine-readable mapping is `index.json`.
- Status `approved-review` means the image is eligible for the default image map.
